  </behavior>
  <action>
    Create `animecix-v2/src/player/tau-protocol.ts` that exports:

    1. `resolveAssetPath(pathname: string, basePath: string): string | null` -- a PURE function (testable without Electron) that:
       - Decodes `pathname` with `decodeURIComponent`
       - If pathname is `/` or empty, maps to `/index.html`
       - Joins with `basePath` via `path.join(basePath, pathname)`
       - Calls `path.resolve()` on the result
       - Verifies the resolved path starts with `path.resolve(basePath)` -- if not, returns `null` (path traversal)
       - Returns the resolved path

    2. `getMimeType(ext: string): string` -- returns MIME type for known extensions:
       - `.html` -> `text/html`
       - `.js` -> `application/javascript`
       - `.css` -> `text/css`
       - `.json` -> `application/json`
       - `.wasm` -> `application/wasm`
       - `.woff2` -> `font/woff2`
       - `.png` -> `image/png`
       - `.ico` -> `image/x-icon`
       - default -> `application/octet-stream`

    3. `registerTauProtocol(): void` -- called AFTER app.whenReady(). Uses `protocol.handle('tau-player', ...)` from Electron:
       - Computes `basePath = path.join(__dirname, '..', 'assets', 'tau-website')`
       - For each request: parse URL, extract pathname, call `resolveAssetPath`
       - If null (traversal), return `new Response('Forbidden', { status: 403 })`
       - Otherwise, return `net.fetch(pathToFileURL(filePath).toString())`

    4. At MODULE TOP LEVEL (not inside a function), export a call to register the scheme as privileged. This MUST be called before app.ready. Export it as a side-effect import or as a separate function `registerTauScheme()`:
       ```typescript
       import { protocol } from 'electron';
       protocol.registerSchemesAsPrivileged([{
         scheme: 'tau-player',
         privileges: {
           standard: true,
           secure: true,
           supportFetchAPI: true,
           stream: true,
           bypassCSP: true,
         },
       }]);
       ```
       CRITICAL: This block MUST execute at import time (module top-level), NOT inside a function, because Electron requires it before app.ready fires.

    Write tests in `animecix-v2/tests/player/tau-protocol.test.ts` that test `resolveAssetPath` and `getMimeType` as pure functions (no Electron mocking needed). Use a temporary directory as basePath in tests.
  </action>
