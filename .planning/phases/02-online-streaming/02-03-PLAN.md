      const row = this.db

        .get(animeId) as { language: string } | undefined;

    2. `parseDeepLinkUrl(url: string): { status: string | null; data: string } | null` -- PURE function:
       - Validates URL starts with 'animecix://'
       - Extracts path after 'animecix://'
       - Must start with 'login' (reject other paths)
       - Splits on 'login' prefix: remaining is `{status|data}` or just `{data}`

       - Validates data does not contain '..' or '/' (prevent path traversal in callback URL)

       - Returns null if data is empty or contains '..' or '/'
       - Returns `https://animecix.tv/secure/short-login/${data}`

    4. `extractDeepLinkFromArgs(args: string[]): string | null` -- PURE function:
       - Finds first arg starting with 'animecix://'
       - Returns it, or null if not found

    5. `handleDeepLink(url: string, webContents: Electron.WebContents): boolean` -- orchestration function:
       - Calls parseDeepLinkUrl
       - If valid, calls buildCallbackUrl
       - If valid callback URL, navigates webContents via `webContents.loadURL(callbackUrl)`
       - Returns true if handled, false if invalid

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OS -> app via animecix:// protocol | Any application can invoke animecix:// URLs -- input is untrusted |
| Deep link data -> webContents navigation | Parsed data becomes part of a navigation URL |
