## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-01 | Information disclosure | Apple ASC API key in CI logs | mitigate | Config reads env vars only; never `console.log` secrets; CI uses `::add-mask::` on decoded key (Plan 04-04) |
| T-4-03 | Tampering | Unsigned-update installation | mitigate | osxSign + osxNotarize guarantee every ZIP delivered through electron-updater is Apple-notarized; electron-updater verifies codesign signature before swap |
| T-4-06 | Elevation of privilege | Accidental public release | mitigate | `draft: true` gate forces manual "Publish release" click in GitHub UI before users receive the update |
| T-4-10 | Spoofing | publisher-github repository mismatch | mitigate | repository owner/name hardcoded to `CaptainSP/AnimeciX-Desktop-Apps`; GITHUB_TOKEN must have access only to this repo |
